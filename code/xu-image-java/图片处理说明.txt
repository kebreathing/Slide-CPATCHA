图片处理工作说明：
1. 把./chris-fronted/img/下的所有图片拷贝至当前目录，并新建目录/img
2. 对刚拷贝的图片进行处理，生成对应的拼图处理图和处理图对应的子拼图
   2-1. 处理图统一放在新文件夹中，文件夹名字在原图文件名上新加标识
   2-2. 子拼图统一放在新文件夹中，文件夹名字在处理图文件名上新加标识
   2-3. 对一个原图生成2-3个处理图
   2-4. 对一个处理图生成1个子拼图
   2-5. 子拼图生成其旋转后的子拼图（正确的子拼图名字后不加后缀，错误的子拼图添加后缀）
3. 格式化代码逻辑，可以分离的操作要分离开来
  3-1. 人性化以及可用性
  3-2. 代码可读性


图片处理例子：
1. 图片原图存储于 /img/目录下
2. 读取 /img/photo1.jpg，对其进行拼图处理
3. 将生成的处理图photo1-puzzle1.jpg存储于/img/puzzle/目录下
4. 将生成的子拼图photo1-puzzle1-sub1.jpg存储于/img/subpuzzle/目录下
5. 对子拼图进行旋转，生成photo1-puzzle1-sub1-1.jpg，同样存储于 /img/subpuzzle/中
6. 上述文件及文件夹的名字可以自由更换，但是一定要存在标识，可以从名字追溯原图、处理图和子拼图
